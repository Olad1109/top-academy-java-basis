## Определение массива в Java
Массив, что это, какое значение он имеет в java!? Представьте себе отсек камеры хранения. Каждый отсек имеет свой номер, в котором хранится некий объект “Вещь”. Или билет в кинотеатре, имеющий номер, вам достаточно дать этот билет контролёру, и он произведёт надрез. Или список пассажиров поезда, за каждым из которых закреплено место с определённым номером. В Java чтобы обрабатывать подобную структуру, то есть множество однородных данных, возникает необходимость в использовании массивов.
## Что такое массив?
Это структура данных, в которой хранятся элементы одного типа. Подобное можно представить, как набор пронумерованных ячеек, в каждую, из которых, можно поместить один из элементов. Доступ к конкретной ячейке осуществляется через её номер, начиная с нуля. Номер элемента в массиве называют индексом, как правило.
## Объявление массива
Массив в Java нужно объявить, подобно объявлению переменной, используя при этом java-синтаксис. См. пример ниже.
```
double[] arrayDouble;
```
В данном случае double - тип переменных, в массиве, что являются числами с плавающей запятой. Таким образом при объявлении массива у него присутствует имя arrayDouble и тип содержимого.
## Создание массива
Создать массив в Java, то есть зарезервировать под него место в памяти, можно с помощью оператора new. Делается это так:
```
arrayDouble = new double[n];
arrayStr = new String[n]
```
На примере выше был использована переменная n для указания количества элементов массива, т. е. её длины. Таким образом, после данной операции мы получаем массив из n-количества элементов, каждый, из которых, равен пустому, если это строка, либо нулевому значению, если это массив чисел.
## Узнать длину массива
Получить доступ к длине массива можно с помощью переменной length. Для примера:
```
int[] myArray = new int[10];
System.out.println(myArray.length);
```
## Инициализация массива и доступ к его элементам
Инициализация массива — это заполнение его конкретными данными (не по умолчанию). Для примера:
```
String[] seasons  = new String[] {"Winter","Spring", "Summer", "Autumn"};
double[] arrayDouble = new double[] {12.09, 10,5, 24.0, 72.0}
```
## Вывод массива в консоли
Вывести элементы массива на экран (то есть, в консоль) можно, например, с помощью цикла for. Глянем на пример ниже.
```
double[] arrayDouble = new double[] {12.09, 10,5, 24.0, 72.0};
for (int aIndex = 0; aIndex < 4; aIndex ++)
System.out.println(arrayDouble[aIndex]);
```
## Одномерные и двумерные массивы
Существует такое понятие, как двумерный массив, состоящий из строк и столбцов, такой массив объявляется и создается следующим образом:
```
int[][] myTwoDimentionalArray = new int [8][8];
```
В этом массиве ровно 64 элемента.
## Полезные методы для работы с массивами
Для работы с массивами в Java есть класс java.util.Arrays (arrays на английском и означает “массивы”). В целом с массивами чаще всего проделывают следующие операции: заполнение элементами (инициализация), извлечение элемента (по номеру), сортировка и поиск.
## Главное о массивах
- Главные характеристики массива: тип помещённых в него данных, имя и длина.

- Последнее решается при инициализации (выделении памяти под массив), первые два параметра определяются при объявлении массива.

- Размер массива (количество ячеек) нужно определять в int

- Изменить длину массива после его создания нельзя.

- Доступ к элементу массива можно получить по его индексу.

- В массивах, как и везде в Java, элементы нумеруются с нуля.

- После процедуры создания массива он наполнен значениями по умолчанию.
