## Структура папок

По умолчанию рабочая область содержит две папки, в которых:

- `src`: папка для хранения исходных кодов
- `lib`: папка для поддержания зависимостей/артефактов

Между тем, скомпилированные выходные файлы по умолчанию будут сгенерированы в папке `bin`, либо `out`.

> Если вы хотите настроить структуру папок, откройте ".vscode/settings.json" и обновите там соответствующие настройки, что справедливо для VS Code. В случае, если у вас Intellij Idea, то открываем директорию .idea, содержащая важные файлы и папки, которые помогут в работе с проектом, включая файлы конфигурации, настройки IDE, сведения о зависимостях, файлы для отладки и т. д.
